# MelihSarikaya_UnityTutorial
## What I Learned
In this tutorial series, I learned the basics of the Unity Editor:
* How to navigate the Scene view and Hierarchy.
* Creating and manipulating GameObjects (Transform, Scale, Rotate).
* Using the Inspector window to modify components.
* Project organization and using Unity Packages.
* Setting up a Unity project with Git and using a proper .gitignore to exclude the Library folder.
